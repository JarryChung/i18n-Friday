# InternationalizatioN Friday 多语言管理平台

![i18n Friday](https://img.shields.io/badge/i18n-Friday-blue) ![Jarry Chung](https://img.shields.io/badge/author-Jarry%20Chung-orange)

## 运行

1. 安装依赖

```shell
yarn boot
```

2. 运行项目

```shell
yarn start
```

3. 访问
   - **Client**: 127.0.0.1:3000
   - **Server**: 127.0.0.1:3001
